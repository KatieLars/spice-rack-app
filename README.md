Spice Rack is a CRUD web app that tracks and manages spices and recipes.

You can access it online at


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spice-rack-app'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install spice-rack-app
